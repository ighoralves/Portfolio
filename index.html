habilidades

HTML5 é a versão mais recente da linguagem de marcação usada para criar páginas web. Ela fornece novos recursos e melhorias em relação às versões anteriores do HTML, como suporte a áudio e vídeo integrado, novas tags semânticas e maior compatibilidade com dispositivos móveis.

CSS3 é a versão mais recente da linguagem usada para estilizar e formatar conteúdo em páginas web. Ela fornece novos recursos, como transições e animações, e melhorias na capacidade de layout responsivo para diferentes tamanhos de tela.

JavaScript é uma linguagem de programação usada para criar interatividade e dinamismo em páginas web. Ela permite adicionar recursos como menus pop-up, animações e validação de formulários, e também é usada em muitas aplicações web e móveis para criar funcionalidades mais avançadas.

Sass é um pré-processador de CSS que permite escrever estilos de forma mais organizada e reutilizável, com recursos como variáveis, mixins e funções. Ele compila o código Sass em CSS tradicional que pode ser usado em páginas web.

Git é um sistema de controle de versão usado para gerenciar e rastrear alterações em arquivos e projetos de software. Ele permite que várias pessoas trabalhem no mesmo projeto ao mesmo tempo, gerenciem conflitos e façam alterações sem afetar o trabalho dos outros.

GitHub é uma plataforma de hospedagem de código-fonte baseada em Git que permite que indivíduos e equipes colaborem em projetos de software. Ele fornece ferramentas de gerenciamento de projetos, controle de versão, revisão de código e integração contínua, tornando mais fácil para os desenvolvedores colaborarem em grandes projetos.
